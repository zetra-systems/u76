# u76 - Universal Base-76 Encoding

u76 is a next-generation encoding standard developed by **Zetta Systems**. It bridges the gap between machine-efficient binary representation and human-readable alphanumeric strings.

## The Vision

* **Higher Density:** More information per character than Base62.
* **Security-First Design:** Formal spec designed for constant-time implementations to prevent side-channel attacks.
* **Modern Alphabet:** A carefully curated 76-character set that is URL-safe and avoids visual ambiguity.
